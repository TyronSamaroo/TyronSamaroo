## Experiments

**Experiment 1** — [ASL-Alpha-NeuralNet](https://github.com/TyronSamaroo/ASL-Alpha-NeuralNet)  
_CNN that reads American Sign Language letters from a webcam. Accessibility + vision._  
**Finding.** data curation beat every architecture change I tried

**Experiment 2** — [Compiler-TinyJ](https://github.com/TyronSamaroo/Compiler-TinyJ)  
_A tiny Java compiler. Lexer → parser → codegen, end to end. From scratch._  
**Finding.** parser is boring; codegen is where you learn how computers work

**Experiment 3** — [Palate](https://github.com/TyronSamaroo/Palate)  
_Full-stack React app for food discovery. Team project._  
**Finding.** modeling taste as latent preference beats explicit tags

**Experiment 4** — [EtsyScraper](https://github.com/TyronSamaroo/EtsyScraper)  
_Web scraper built with only the Java JDK. No third-party libs. Pure grit._  
**Finding.** constraint-driven projects are the fastest way to understand a stack

**Experiment 5** — [SearchEngine](https://github.com/TyronSamaroo/SearchEngine)  
_Tiny search engine: indexing, ranking, query parser._  
**Finding.** classical ranking still beats cosine similarity for explainability

**Experiment 6** — [Stock_App_Flask](https://github.com/TyronSamaroo/Stock_App_Flask)  
_Flask stock tracker + portfolio views. Market data + caching._  
**Finding.** 90% of the work ended up in the caching layer, not the app
